### About

First-year B.Tech (IT) student at MAIT Delhi building at the boundary of systems programming, decentralized identity, and compiler infrastructure. I ship across the full stack — from low-level Rust and WASM runtimes to SSI portals and RISC-V toolchains — driven by first-principles thinking and a bias toward minimal, correct engineering.

- 🦀 **Primary:** Rust — systems, WASM, CLI, QUIC
- 🌐 **Web:** TypeScript, Next.js, React, Node.js
- 🧠 **ML:** Python, Neural Nets, Signal Processing
- 📍 New Delhi, India
